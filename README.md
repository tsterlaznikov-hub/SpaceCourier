# Космический Курьер — стелс-аркада

**Курс Ulearn, УрФУ 2025**  
2D-игра на Windows Forms (C#) по паттерну **MVC**

## Геймплей
- WASD — движение
- **Shift** — маскировка (невидимость, тратит энергию)
- **Space** — щит (блокирует урон)
- **E** — бросить приманку (отвлекает патрули)
- **Esc** — пауза

**Цель:** Долететь от станции до планеты, не будучи пойманным патрулями Империи.

## Особенности
- Патрули видят на **360°**, помнят последнюю позицию цели
- Приманка с **белым маяком** — отвлекает всех в радиусе
- Полная погоня, координация патрулей
- Красивый звёздный фон с параллаксом
- Чистый MVC, готово к тестам
